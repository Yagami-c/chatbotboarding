### 三、组件详细设计

#### 3.1 用户信息卡片（顶部）

**背景区：**
| 属性 | 值 |
|------|-----|
| 高度 | 164px |
| 背景 | 渐变色 `linear-gradient(135deg, #2ECC71 0%, #1A8A4A 100%)` |
| 圆角 | 底部圆角 16px（与页面背景融合） |

**用户信息布局：**
```
┌─────────────────────────────────────────────────────────────┐
│  ┌─────────────┐                                           │
│  │   👤         │  张三                      编辑资料 →    │
│  │  头像 80×80  │  会员 ID: 88472910                      │
│  │  带白色边框  │                                          │
│  └─────────────┘                                           │
│  ─────────────────────────────────────────────────────────── │
│  400              12               5                       │
│  可用积分          训练天数         评估报告                │
└─────────────────────────────────────────────────────────────┘
```

| 元素 | 规格 |
|------|------|
| 头像 | 80×80px，圆形，2px 白色边框，背景浅灰色占位 |
| 用户昵称 | 字号 20px，字重 600，颜色 #FFFFFF |
| 会员 ID | 字号 13px，颜色 rgba(255,255,255,0.8) |
| 编辑资料 | 字号 13px，颜色 #FFFFFF，带右箭头 → |
| 统计数据 | 白色文字，数字 20px/700，标签 13px/400 |
| 统计分割线 | 1px 竖线，白色半透明 |

**交互：**
- 点击头像 → 更换头像（系统相册）
- 点击编辑资料 → 跳转资料编辑页
- 点击统计数字 → 跳转对应详情页


#### 3.2 功能菜单列表

所有菜单项统一规格：

| 属性 | 值 |

| 高度 | 56px |
| 背景 | #FFFFFF |
| 左右内边距 | 16px |

| 序号 | 图标 | 标题 | 辅助信息（右） | 说明 |
