### 四、首次登录 Onboarding（精简为智能模式提示）

首次进入 App 时，展示一个**轻提示卡片**（非全屏引导）：

```
┌─────────────────────────────────────────────┐
│  💡 让小瑞陪你康复                          │
│                                              │
│  开启「智能模式」，小瑞将全程引导你：        │
│  • 每日评估与方案推荐                       │
│  • 设备使用陪伴与实时反馈                   │
│  • 强度自动优化与科普推送                   │
│                                              │
│  [✨ 开启智能模式]   [🙅 暂不开启]          │
│                                              │
│  可在「我的-设置」中随时调整                 │
└─────────────────────────────────────────────┘
```

- 卡片在首页底部以横幅形式出现（非全屏遮罩）
- 选择后存入 localStorage，不再重复显示
- 智能模式开启 → 小瑞主动推送；关闭 → 小瑞被动响应


### 五、小瑞核心流程（5步Pipeline）

```
步骤条（仅点击「启动设备」后显示）：
✅ 小瑞了解你 → 基本信息 & 评估
⏳ 推荐方案 → 生成初始强度
⏸ 陪伴治疗 → 设备控制（遥控器独立）
⏸ 记录变化 → 使用反馈
⏸ 持续优化 → 调整下次强度
```

#### Day1 评估流程（聊天卡片形式）

| 步骤 | 内容 | 控件 |
|------|------|------|
| 1 | 称呼 | 文本输入 |
| 2 | 性别 | 男/女/其他 |
| 3 | 年龄 | 20↓/20-40/40-60/60↑ |
| 4 | 不适时长 | 不到1月/1-3月/3-6月/6月-1年/1年以上/无 |
| 5 | 安全筛查 | 多选+补充 → 选中风险项弹窗警告 |
| 6 | 膝盖紧度 | 单选+频率 |
| 7 | 触发动作 | 多选+补充 |
| 8 | 疼痛程度 | 0-4级单选+补充 |

#### 推荐方案卡片

```
📍 最困扰动作：下楼梯  ⭐⭐⭐☆ (3/4)
🌸 推荐：舒缓模式
├── 压力 125 mmHg
├── 工作 30秒 / 休息 10秒 / 5个循环
└── 每天1-2次，每次约3分钟
[📹 教学视频]  [🔌 启动设备]
```

#### Day2-6 日常

| 步骤 | 内容 |
|------|------|
| 了解 | 今天膝盖感觉如何？（更舒服/没变化/更不舒服） |
| 推荐 | 展示调整后方案 |
| 陪伴 | 使用前须知 + 遥控器控制 |
| 记录 | 未完成原因（如中途结束） |
| 优化 | 今日总结（更好/差不多/更差） |

#### Day7 复评

**前置提示页（全屏居中卡片）：**
```
📊 阶段评估提醒
已完成6天训练，今天进行第7天复评
预计耗时：3-5分钟
[开始复评 →]
```

**复评流程：**
1. 最困扰动作还是「下楼梯」吗？（是/否）
2. 现在不适程度？（0-4级）
3. 7天整体感觉？（改善/差不多/变差）
4. 阶段总结卡片（四种结果）

#### 阶段总结四种结果

| 类型 | 图标 | 对比展示 |
|------|------|----------|
| 明显改善 | 🎉 | ⭐⭐⭐☆☆ → ⭐☆☆☆☆ |
| 轻度改善 | 🎉 | ⭐⭐⭐☆☆ → ⭐⭐☆☆☆ |
| 基本稳定 | 📊 | ⭐⭐⭐☆☆ → ⭐⭐⭐☆☆ |
| 状态下降 | ⚠️ | ⭐⭐⭐☆☆ → ⭐⭐⭐⭐☆ |


### 六、中途结束询问

**触发条件：** 用户点击「结束」按钮 / 设备异常停止 / Day2-6 未完成治疗

**弹窗内容：**
```
⏹ 训练已结束
今天没有完成预定治疗，主要原因是什么？
☐ 忘记了
☐ 没时间（注明原因）______
☐ 效果不明显（注明依据）______
☐ 使用不舒服（注明表现）______
☐ 其他（注明）______
[确定]
```

- 选择「忘记/没时间/效果不明显」→ 鼓励话术：「持之以恒，有助康复 💪」
- 选择「使用不舒服」→ 自动降级 1 级


### 七、悬浮球遥控器（独立硬件控制）

#### 视觉规范

| 属性 | 值 |
|------|-----|
| 尺寸 | 56×56px |
| 位置 | 右侧贴边，可垂直滑动 |
| 背景 | `rgba(46,204,113,0.2)` + `backdrop-filter: blur(8px)` |
| 边框 | `1px solid rgba(255,255,255,0.3)` |
| 图标 | 28×28px，白色线性 |

#### 状态表

| 状态 | 背景 | 图标 | 指示灯 | 动画 |
|------|------|------|--------|------|
| 未连接 | `rgba(200,200,200,0.2)` | 🔌 | 灰色 | 无 |
| 已连接待机 | `rgba(46,204,113,0.25)` | ✅ | 绿色 | 无 |
| 运行中 | `rgba(46,204,113,0.35)` | ⚙️ | 绿色呼吸 | 旋转 |
| 暂停 | `rgba(243,156,18,0.25)` | ⏸ | 橙色 | 停止 |
